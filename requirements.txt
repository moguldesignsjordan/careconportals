# Care General Construction Portal - Project Requirements

## Technical Environment
- Node.js: v18.0.0 or higher
- Package Manager: npm (v9+) or yarn
- Build Tool: Vite (recommended)
- Browser Support: Modern Evergreen Browsers (Chrome 90+, Edge 90+, Safari 15+, Firefox 90+)

## Core Tech Stack
- Frontend: React 19.2.3 (TypeScript)
- Styling: Tailwind CSS 3.4+
- Icons: Lucide React
- Charts: Recharts
- Backend/Auth: Firebase (Authentication & Firestore)

## Authentication Requirements
- Provider: Google Identity Platform (via Firebase)
- Role-based Access Control (RBAC):
    - ADMIN: Full system oversight
    - CONTRACTOR: Site logs and progress management
    - CLIENT: Progress tracking and document access

## Functional Features
1. Dashboard: Role-specific views with progress visualizations.
2. Project Management: Creation, assignment, and tracking of construction stages.
3. Messaging System:
    - Real-time communication between all parties.
    - User tagging (@mentions) for team members.
    - Context-aware chats (Project Leads/Admins).
4. Site Updates: Multimedia log entries (Text/Photos) for field progress.
5. Document Portal: Centralized storage for Contracts, Blueprints, Permits, and Invoices.
6. Notification Center: Real-time alerts for project milestones and messages.

## Design Specifications
- Brand Color: #F15A2B (Care Orange)
- Primary UI Tone: #1A1A1A (Deep Dark)
- Backgrounds: #F9FAFB (Gray 50)
- Font Stack: Inter (Sans-serif)
- Aesthetics: High-contrast, clean typography, "Glassmorphism" headers.
