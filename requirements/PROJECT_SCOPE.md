# Project Scope: Care General Construction Portal

## Overview
A comprehensive construction management dashboard designed to bridge the communication gap between contractors, clients, and management. The portal serves as the single source of truth for project progress, documentation, and real-time coordination.

## Core Objectives
- **Transparency**: Clients should see exactly where their money is going and what stage their project is in.
- **Efficiency**: Contractors need a streamlined way to log site updates and access blueprints/permits in the field.
- **Accountability**: Real-time messaging and update logs create a permanent record of site activities.

## User Roles
1. **Admin (Management)**
   - Oversight of all active projects.
   - Resource allocation (assigning contractors to clients).
   - Global directory management and document control.
2. **Contractor (Field Operations)**
   - Daily site logs and progress updates.
   - Photo/Media attachment for proof of work.
   - Access to technical documents (blueprints).
3. **Client (Homeowner)**
   - Visual progress tracking (percentage bars).
   - Direct line to site leads.
   - View/Download invoices and contracts.
