# User Stories

## Admin Stories
- **US-A1**: As an Admin, I want to create a new project and link a client to a contractor so that the build process can begin.
- **US-A2**: As an Admin, I want to view a global financial overview (Budget vs. Spent) to ensure project profitability.
- **US-A3**: As an Admin, I want to manage the user directory to add new specialized contractors to the team.

## Contractor Stories
- **US-CO1**: As a Contractor, I want to post a site update with a timestamp so that the client and office stay informed.
- **US-CO2**: As a Contractor, I want to @mention an admin in a message to quickly resolve material shortages or permit issues.
- **US-CO3**: As a Contractor, I want to update the progress slider so that the client can see the transition between phases (e.g., Demo to Rough-in).

## Client Stories
- **US-CL1**: As a Client, I want to see a live progress bar so that I feel confident in the project timeline.
- **US-CL2**: As a Client, I want to receive notifications when a new site update is posted by the contractor.
- **US-CL3**: As a Client, I want to access my signed contract and blueprints in one place to avoid searching through emails.
